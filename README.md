# vivadecora-frontend-challenge

## Project setup

### Clone the repo

```
git clone 'this project'
```
### Go to the paste

```
cd 'folder-name'
```

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
